CWDog
=====

Morse code trainer.

See wiki.

Provides real time on the fly morse text to sound card audio with variation in modulation tone and speed.  High quality
sin wave audio without clipping and a simple keyboard based iambic keyer for practice.  Straight keyer had too much lag
in the audio system to attempt that.
